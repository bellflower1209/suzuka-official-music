ECLYPSE chapter two

The code is broken.
Now the red moon rises.

「RED MOON // RISING」は、ECLYPSEの2nd Single。壊れたコード、赤く染まる月、そして封じられていた運命が動き始める光景を描きます。

崩壊した近未来都市を覆う赤い光の中、それぞれの傷と宿命を背負った5人が、逃れられない未来へ立ち向かうダークSF作品です。

デビューシングル「SHADOW//CODE」で掲げた運命を書き換える物語から、コードが壊れた先のECLYPSE第2章へ。黒と赤のビジュアルが、覚醒と再起動の瞬間を映し出します。
