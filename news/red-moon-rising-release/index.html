ECLYPSE
「RED MOON // RISING」公開

赤い月とともに始まる、ECLYPSE第2章。壊れたコードの先で、5人の運命が再び動き始めます。

記事公開日
2026年7月20日
楽曲公開日
2026年7月18日
Author
SUZUKA

01 / Story

The code is broken.
Now the red moon rises.

2026年7月18日、ECLYPSEの2nd Single「RED MOON // RISING」がSUZUKA公式YouTubeチャンネルで公開されました。

壊れたコード。赤く染まる月。そして、封じられていた運命が動き始める。公式MVとジャケットは、崩壊した近未来都市を背景に、黒と赤の光の中で立つ5人を描いています。

デビューシングル「SHADOW//CODE」で示された“運命を書き換える”物語は、コードが壊れた先へ。赤い月の光が、ECLYPSEの覚醒と再起動を映し出します。

02 / Words

五つの心、ひとつの運命。

FIVE HEARTS.
ONE DESTINY.

公式ジャケットとYouTube説明文に記された言葉です。5人がそれぞれの傷と宿命を背負いながら、同じ未来へ立ち向かうECLYPSEの核を短く示しています。
